Final Results

In alignment with our methodology. We created a solution that has both a data collection component and a data visualization aspect. We created a survey that allows for individuals to report/record hidden address information. This survey format is like a form of citizen science. Since it’s difficult to find these hidden addresses for various reasons there is no cookie cutter method that can be used. Instead citizen science allows individuals to send in reports if they believe an address is misrecorded.

Reflect on Problem Statement

Problem Statement Solution
Ideal: User friendly centralized web platform that allows individuals to report hidden apartment addresses on a 2D map. The Survey123 addresses the reporting aspect. The Dashboard & Experience builder can displace these addresses on a 2D map
Proposal: Use a survey web application to record user reported hidden address data. This information can then be transferred to a database and allows city workers to confirm the reporting’s before updating their own database. Survey123 allows individuals to self report hidden address information. This information can then be transferred to city workers through a process that will be addressed in Next Steps
Reality: It is currently difficult to determine hidden addresses. This makes it difficult for EMS services to provide help to unidentified locations requiring them to use valuable time to search for a unit. With a more accurate database through hidden address reporting there should be better outcomes for emergency services.

Next Steps

Next Steps The Survey’s quality assurance and quality control could be improved. For example: more restrictions for the fields e.g. letters only. Also further formatting of the address field so the first letter of a word is always capitalized. Webhooks can be created to send notifications via email or SMS to individuals that are responsible for certain areas (e.g. wards). This ensures that a municipality employee is aware when a new location is submitted and allows a spreadsheet to keep up with follow up. Explore open source options. While ESRI products are widely used it is not guaranteed because of the cost associated with the licenses so exploring other options that are more accessible can widen the application of this type of solution
